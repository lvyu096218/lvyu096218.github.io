自卑与超越读后感

20223-4-5
0  -  0m
学习笔记
人文社科
个人发展

# 前言

“生命的意义”都有一个共同标志——能够为众人所分享，为他人所接受。它可以成为解决他人问题的一种样本，它将个人意义建立在对他人生命的贡献之上。一想到我所做的一切于他人有益，我就真的很难再自卑下去。

书需反复阅读，方能体会其中的真理！摆脱自卑，超越自我！

# 关于意义

人类为了生存发展出了与人合作的能力。事实上宇宙并没有本质的对错，人类社会的道德评价是基于人类产生的。这些道德，各种观念的产于人类的合作关系。从野蛮时代合作代表着更大几率的生存。现在的合作意味着一个人是否能够融入社会，发展成为我们所谓健康的人类。心理疾病的产生也是因为对合作的不擅长，不正确的认识。

我们评价某件事是不是正确的，某个人是不是健康完美。应该从人类的角度出发。做某件事是否有利于自己，或者有利于他人。对自己他人有害的事情必然不受认可。也就是我们所说的不正确的事情。对自己有益对他人有害的事情在这个时代也被定义为错误，这类错误经常会蒙蔽个人的双眼，让人发展畸形，产生不健康的人格，最终丧失掉对自己有利的部分，发展成为对自己无利社会无利的事情。

# 怎样面对这些问题

阿德勒教授通过大量的案例分析总结，挖掘内外的异同，总结出了个人健康发展的建议。 首先我们明确人个体之间存在差异，但是这些差异还不足以放大到未来巨大的差异。这些差异通过心理的作用从而发出巨大的威力，产生巨大的影响。从很多身体有缺陷的人取得辉煌的成就，或者就此一蹶不振。都是心理在作祟，阿德勒教授给出了令人信服的观点。 这些人通过最底层的对于生活的理解不同，对待自身的缺陷产生了不同的后果。所有人都在用自己的方式解决困难。那些最终成功的人不过是因为对于解决问题的看法更正确。很多因此一蹶不振的人因为自己不正确的认知产生了错误的解决方式于是被生活击跨。当面对问题时让我们仔细想想什么才能算是正确的面对方式。

# 自卑是常态

基于这样的理论阿德勒教授从几个方面探讨了这些人面对的困境到底基于怎样不正确的底层思想。 自卑是一个人正常的发展特性，书中解释自卑本身不是变态的，他是人类地位之所以增进的原因。评估一个人的自卑是否正确便要从上面的评估来入手，当自卑发生时是否能勇敢面对，这些自卑是否能让他更好的发展自己和他人。自卑存在于人生各个阶段。 追求优越是一种健康的方式，但是神经病患者，对生活失望的人都是没能正确的知道怎么才能追求到优越，他们用错误的底层思维给出追求优越变态的解释，这样的解释没有合作概念，没有为自己和他人带来利益。 人类的行为模式都是底层思维的体现，他怎样理解世界于是就会有怎样的行为模式。

# 起源

作者还从早期记忆和梦来分析一个人的底层逻辑，指出错误。 再者作者分析了家庭，学校怎样塑造出来了一个人底层的对于世界的认识。尤其家庭之中，父母，兄弟姐妹怎么最早塑造了这些思想。从夫妻之间不能偏心，在孩子没有学到合作意识前不能太亲密，孩子对于母亲最开始的信任如何建立，再如何健康的到父亲，到兄弟姐妹，到家庭外的其他人。这些步骤缺一不可。都是孩子建立健康心理的重要步骤。

# 最后

最后对于犯罪，青春期，职业选择作者都给出了分析和指导。都是基于要与人合作的观念，做利于自己和他人事出发，来讲述哪些到底是追求，哪些到底是错误。
